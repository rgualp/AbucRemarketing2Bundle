For using importing_data.sql:

This file is a sql script. 
It suppose that both databases are in the same server. 
For running this in server, you must rename the name mypalada_mycp to the name of the new database, 
and rename old_mypaladar_mycp to the name of the old database.

This file import data in destination tables, ownership tables, faq tables, lang tables, province table, municipality table and 
albums tables.

For using importing_photos_files.php:
This file is a PHP script.
Before using it you must configure images path in old and new website and to configure old database access credentials.
The file is commented and the name of the variables to configure are understandable.
